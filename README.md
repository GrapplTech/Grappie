## What is Grappie?

Grappie is a **browser-only, BYOK (Bring Your Own Key) AI interview simulator** that turns your resume into a personalised mock interview in under 30 seconds.

- **Upload your resume** → Grappie reads it entirely in your browser (no server upload)
- **Pick your interview type** → Technical, HR, Behavioural, or Mixed
- **Set a target role + company** → Questions are hyper-tailored to the actual employer
- **Answer by voice** → Real-time speech-to-text transcription
- **Get a full report** → Scores, strengths, weaknesses, action items — downloadable as HTML

Zero accounts. Zero subscriptions. Zero data sent to any server except the text prompt to OpenRouter.

## Getting Started

### Option 1 — Open Directly (No Server Needed)

```bash
git clone https://github.com/your-username/grappie.git
cd grappie
open index.html          # macOS
# xdg-open index.html   # Linux
# double-click in Explorer on Windows
```

> **Note:** Camera/mic access may require HTTPS or localhost in some browsers. Use Option 2 if you hit permission errors.

### Option 2 — Serve Locally (Recommended)

```bash
# Python (built-in)
python3 -m http.server 8080

# Node.js
npx serve .

# Then visit: http://localhost:8080
```

### Option 3 — Deploy to GitHub Pages

1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root directory
4. Live at `https://your-username.github.io/grappie`

## How to Get Your API Key

Grappie uses [OpenRouter](https://openrouter.ai) — a single gateway to 200+ AI models with a generous free tier.

1. Go to [openrouter.ai](https://openrouter.ai) and create a free account
2. Sign in with Google, GitHub, or email
3. Navigate to **Settings → API Keys**
4. Click **Create Key** and copy it (starts with `sk-or-v1-`)
5. Paste it into Grappie's setup page and click **Validate**

**Free models available:** DeepSeek Chat, Gemma 3 12B, Phi-3 Mini — no credit card needed.

## Usage Guide

1. **Upload Resume** — drag your PDF/DOCX/TXT onto the upload zone
2. **Select Interview Type** — Technical, HR Round, Behavioural, or Mixed
3. **Enter Target Role** — e.g. "Senior Frontend Engineer"
4. **Enter Target Company** — questions will be tailored to their culture
5. **Validate API Key** — paste your OpenRouter key and click Validate
6. **Choose AI Model** — DeepSeek Chat is recommended (fast + free)
7. **Set Duration** — 5 to 30 minutes
8. **Grant Camera + Mic** — one browser permission prompt covers both
9. **Begin Interview** — answer 8 questions by voice
10. **Get Your Report** — download the HTML report with scores and action items

**During the interview:**
- Press **Space** or tap the mic button to toggle recording
- Click **Submit Answer** when done speaking
- Click **Skip** to move on, **Repeat** to hear the question again
- A 15-second think timer precedes each question before the mic opens

## Tech Stack

| Layer | Technology | Why |
|---|---|---|
| Markup | HTML5 | Semantic, no build step needed |
| Styling | Custom CSS (design tokens) | Full control, no Tailwind purge needed |
| Logic | Vanilla JavaScript (ES2020) | Zero dependencies, fast, auditable |
| PDF parsing | [pdf.js](https://mozilla.github.io/pdf.js/) v3.11.174 | Best in class, Mozilla-maintained |
| DOCX parsing | [mammoth.js](https://github.com/mwilliamson/mammoth.js) v1.6.0 | Clean plain-text extraction |
| AI backend | [OpenRouter](https://openrouter.ai) | Access to 200+ models, free tier |
| Speech input | Web Speech API (SpeechRecognition) | Browser-native, no cost |
| Speech output | Web Speech API (SpeechSynthesis) | Browser-native TTS |
| Camera | MediaDevices.getUserMedia | Browser-native, one permission prompt |
